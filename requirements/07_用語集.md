# 07. 用語集

本ドキュメントで使用する用語の定義。

| 用語 | 英語 | 定義 |
|------|------|------|
| 製造指示 | Work Order / Manufacturing Order (MO) | 特定の製品を製造するための指示。数量・工程・日程を含む |
| 生産計画 | Production Plan | 期間内に何をいくつ製造するかの計画 |
| 工程 | Process / Operation | 製造における作業単位（例：加工・組立・検査） |
| ルーティング | Routing | 品目を製造する際の工程の順序・標準時間の定義 |
| BOM | Bill of Materials（部品表） | 製品を構成する原材料・部品とその必要数量の一覧 |
| ロット | Lot | 同一条件で製造・調達された製品・原材料のまとまり |
| シリアル番号 | Serial Number | 個体を一意に識別する番号 |
| トレーサビリティ | Traceability | 製品の製造・流通履歴を追跡できること |
| 引当 | Allocation / Reservation | 在庫を特定の製造指示・出荷指示に予約すること |
| 棚卸 | Physical Inventory | 実際の在庫数量を確認し帳簿と照合する作業 |
| FIFO | First In First Out（先入れ先出し） | 先に入庫したものから先に出庫する在庫管理方式 |
| 受入検査 | Incoming Inspection | 購買・外注品が入荷した際に行う品質検査 |
| 工程内検査 | In-Process Inspection | 製造工程の途中で行う品質検査 |
| 完成品検査 | Final Inspection | 出荷前に行う最終品質検査 |
| 特採 | Conditional Acceptance | 基準に合格しない品目を条件付きで使用・出荷すること |
| 安全在庫 | Safety Stock | 需要変動・調達リードタイムに対応するための最低限の在庫量 |
| リードタイム | Lead Time | 発注から入荷・製造開始から完了までの所要時間 |
| MES | Manufacturing Execution System（製造実行システム） | 製造現場の実績収集・管理を行うシステム |
| ERP | Enterprise Resource Planning（統合基幹業務システム） | 企業全体の業務を統合管理するシステム |
| SCM | Supply Chain Management（サプライチェーン管理） | 調達から出荷までの供給連鎖の管理 |
| QC | Quality Control（品質管理） | 製品・サービスの品質を維持・改善する活動 |
| KPI | Key Performance Indicator（重要業績評価指標） | 目標達成度を測定する指標 |
| UAT | User Acceptance Testing（受入テスト） | システム利用者が実施する最終確認テスト |
| RTO | Recovery Time Objective（目標復旧時間） | 障害発生後にシステムを復旧させるまでの目標時間 |
| RPO | Recovery Point Objective（目標復旧時点） | 障害発生時にどの時点のデータまで復元するかの目標 |
