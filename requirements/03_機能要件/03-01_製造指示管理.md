# 03-01. 製造指示管理

## 概要

製造指示（Work Order）の発行・配信・確認・実績紐付けを管理する機能。

---

## 機能一覧

| 機能ID | 機能名 | 概要 | 優先度 |
|--------|--------|------|--------|
| MO-001 | 製造指示発行 | 生産計画から製造指示を生成・発行する | 必須 |
| MO-002 | 製造指示照会 | 発行済み製造指示の一覧・詳細を確認する | 必須 |
| MO-003 | 製造指示変更 | 数量・日程・工程の変更・修正を行う | 必須 |
| MO-004 | 製造指示キャンセル | 発行済み製造指示をキャンセルする | 必須 |
| MO-005 | 製造指示配信 | 現場端末（タブレット等）へ製造指示を通知する | 必須 |
| MO-006 | 製造指示印刷 | 製造指示書・作業指示書をPDF/印刷出力する | 必須 |
| MO-007 | 進捗ステータス管理 | 製造指示の進捗状態（未着手・進行中・完了）を管理する | 必須 |
| MO-008 | 実績紐付け | 工程実績・品質実績を製造指示に紐付ける | 必須 |
| MO-009 | 製造指示複写 | 既存の製造指示を複写して新規作成する | 推奨 |
| MO-010 | バッチ発行 | 複数の製造指示を一括で発行する | 推奨 |

---

## 機能詳細

### MO-001 製造指示発行

**目的：** 生産計画または手動入力に基づき、製造指示を生成・発行する。

**入力データ：**

| 項目 | 必須 | 説明 |
|------|------|------|
| 製品コード | ○ | 製造する製品のマスタコード |
| 製造数量 | ○ | 指示数量 |
| 製造開始予定日 | ○ | 着手予定日 |
| 完了予定日 | ○ | 完了予定日 |
| 担当工程 | ○ | 対象工程（複数工程可） |
| ロット番号 | ○ | 製造ロット番号（自動採番または手動） |
| 作業指示コメント | - | 特記事項・特別指示 |
| 参照図番 | - | 図面番号・仕様書番号 |

**処理内容：**

1. 入力値のバリデーション
2. 製造ロット番号の採番（自動採番の場合）
3. 対象工程の展開（工程マスタより）
4. 必要原材料の引当確認
5. 製造指示レコードの登録
6. 製造指示番号の採番・返却
7. 現場端末への通知（MO-005）

**出力データ：**

- 製造指示番号
- 製造指示書（PDF）

**業務ルール：**

- 在庫が引当不可の場合、警告を表示するが発行は可能とする（要確認）
- 製造指示番号の採番ルール：`MO-YYYYMMDD-NNNN`（例：MO-20260307-0001）

**画面：** 製造指示発行画面

---

### MO-003 製造指示変更

**変更可能なステータス：**

| 変更前ステータス | 変更可否 |
|----------------|---------|
| 発行済（未着手） | ○ |
| 着手中 | 一部可（数量変更不可） |
| 完了 | × |
| キャンセル | × |

**変更可能項目：**

- 製造数量（未着手のみ）
- 完了予定日
- 担当者・担当工程
- 作業指示コメント

---

### MO-007 進捗ステータス管理

**ステータス遷移：**

```
発行済（未着手）
  |
  v（着手操作）
着手中
  |
  v（完了操作）
完了
```

```
発行済（未着手） or 着手中
  |
  v（キャンセル操作）
キャンセル
```
