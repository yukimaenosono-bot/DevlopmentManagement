# 03-04. 品質管理

## 概要

受入検査・工程内検査・完成品検査の記録、不良管理、トレーサビリティを管理する機能。

---

## 機能一覧

| 機能ID | 機能名 | 概要 | 優先度 |
|--------|--------|------|--------|
| QC-001 | 受入検査登録 | 購買品・外注品の受入検査結果を記録する | 必須 |
| QC-002 | 工程内検査登録 | 製造工程中の検査結果を記録する | 必須 |
| QC-003 | 完成品検査登録 | 完成品の出荷前検査結果を記録する | 必須 |
| QC-004 | 不良登録・管理 | 不良発生の記録・原因・対策管理 | 必須 |
| QC-005 | 不良品処置 | 不良品の処置（手直し・廃棄・特採）を記録する | 必須 |
| QC-006 | トレーサビリティ照会 | ロット・製造指示をキーに前後工程を追跡する | 必須 |
| QC-007 | 品質統計・分析 | 不良率・工程能力等の集計・グラフ表示 | 推奨 |
| QC-008 | 検査基準マスタ管理 | 品目・工程ごとの検査基準を管理する | 必須 |
| QC-009 | 検査成績書出力 | 検査結果をPDFで出力する | 推奨 |

---

## 機能詳細

### QC-001〜003 検査登録（共通仕様）

**入力データ：**

| 項目 | 必須 | 説明 |
|------|------|------|
| 検査日時 | ○ | 検査実施日時 |
| 品目コード | ○ | 検査対象品目 |
| ロット番号 | ○ | 検査対象ロット |
| 製造指示番号 | - | 紐付く製造指示 |
| 検査員 | ○ | 検査実施者 |
| 検査数量 | ○ | 検査した数量 |
| 合格数量 | ○ | 合格数量 |
| 不合格数量 | ○ | 不合格数量（不良数） |
| 検査項目結果 | ○ | 各検査項目の測定値・合否 |
| 判定結果 | ○ | 合格 / 不合格 / 条件付合格 |
| 備考 | - | 特記事項 |

---

### QC-004 不良登録・管理

**不良情報：**

| 項目 | 必須 | 説明 |
|------|------|------|
| 不良発生日時 | ○ | - |
| 不良発生工程 | ○ | - |
| 不良区分 | ○ | 外観・寸法・機能・その他 |
| 不良内容 | ○ | 具体的な不良内容の説明 |
| 不良数量 | ○ | - |
| 推定原因 | - | 発生原因の分析 |
| 是正処置 | - | 再発防止策 |

---

### QC-006 トレーサビリティ照会

**追跡方向：**

- **フォワードトレース：** 原材料ロット → 使用した製造指示 → 完成品ロット → 出荷先
- **バックワードトレース：** 完成品ロット → 使用原材料ロット → 仕入先

**照会キー：**

- ロット番号
- 製造指示番号
- 製品シリアル番号（シリアル管理品目の場合）

---

### QC-005 不良品処置区分

| 処置区分 | 説明 |
|---------|------|
| 手直し | 修正して合格品として使用 |
| 廃棄 | 廃棄処分 |
| 特採 | 基準外だが特別承認で使用 |
| 返品 | 仕入先へ返品（受入不良の場合） |
| 保留 | 処置待ち |
