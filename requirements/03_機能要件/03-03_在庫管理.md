# 03-03. 在庫管理

## 概要

原材料・仕掛品・完成品の在庫を管理する機能。入庫・出庫・棚卸・在庫照会を含む。

---

## 機能一覧

| 機能ID | 機能名 | 概要 | 優先度 |
|--------|--------|------|--------|
| INV-001 | 入庫登録 | 原材料・完成品の入庫を記録する | 必須 |
| INV-002 | 出庫登録 | 原材料の払出・製品の出庫を記録する | 必須 |
| INV-003 | 在庫照会 | 品目・ロット・棚番ごとの在庫数量を確認する | 必須 |
| INV-004 | 在庫移動 | 倉庫間・棚間の在庫移動を記録する | 必須 |
| INV-005 | 棚卸 | 棚卸数量の入力・実在庫と帳簿在庫の差異確認 | 必須 |
| INV-006 | 在庫引当 | 製造指示に対して原材料を引当する | 必須 |
| INV-007 | 安全在庫アラート | 在庫が安全在庫を下回った場合に通知する | 推奨 |
| INV-008 | 在庫履歴照会 | 品目・ロットごとの入出庫履歴を参照する | 必須 |
| INV-009 | バーコード/QRスキャン | ハンディターミナルによるスキャン入力対応 | 必須 |

---

## 機能詳細

### INV-001 入庫登録

**入庫区分：**

| 入庫区分 | 説明 |
|---------|------|
| 購買入庫 | 外部調達品の入庫 |
| 製造完成入庫 | 製造完了品の完成品入庫 |
| 返品入庫 | 出庫済み品の返品 |
| その他入庫 | 上記以外 |

**入力データ：**

| 項目 | 必須 | 説明 |
|------|------|------|
| 品目コード | ○ | 入庫品目 |
| 入庫数量 | ○ | 入庫数量 |
| 入庫区分 | ○ | 上記区分 |
| ロット番号 | ○ | 品目ロット番号 |
| 入庫日 | ○ | 入庫日付 |
| 保管場所（棚番） | ○ | 格納先棚 |
| 仕入先コード | - | 購買入庫の場合 |
| 有効期限 | - | 期限管理品目の場合 |

---

### INV-006 在庫引当

**引当ロジック（デフォルト）：**

- FIFO（先入れ先出し）を基本とする
- 有効期限が近いロットを優先する設定も可能

**引当ステータス：**

| ステータス | 説明 |
|-----------|------|
| 引当済 | 製造指示に予約されている在庫 |
| 実出庫済 | 実際に払い出し済みの在庫 |

---

### INV-005 棚卸

**棚卸フロー：**

```
棚卸開始宣言（在庫ロック）
  |
  v
棚卸数量入力（ハンディ or 画面）
  |
  v
差異確認（帳簿在庫 vs. 実地在庫）
  |
  v
差異承認・調整入力
  |
  v
棚卸完了（在庫ロック解除）
```

---

## 在庫区分

| 在庫区分 | 説明 |
|---------|------|
| 利用可能在庫 | 通常使用可能な在庫 |
| 引当済在庫 | 製造指示に引当されている在庫 |
| 検査中在庫 | 入荷検査中で未判定の在庫 |
| 保留在庫 | 不良・問題品として保留中の在庫 |
| 廃棄在庫 | 廃棄処分が決定した在庫 |
