# 03-05. 工程管理

## 概要

製造工程の進捗・実績（着手・完了・作業時間・設備稼働）を収集・管理する機能。

---

## 機能一覧

| 機能ID | 機能名 | 概要 | 優先度 |
|--------|--------|------|--------|
| WO-001 | 工程着手登録 | 工程の作業開始を記録する | 必須 |
| WO-002 | 工程完了登録 | 工程の作業完了・実績数量を記録する | 必須 |
| WO-003 | 工程進捗照会 | 製造指示・工程ごとの進捗状況をリアルタイム確認 | 必須 |
| WO-004 | 作業実績入力 | 作業時間・使用設備・作業者を記録する | 必須 |
| WO-005 | 設備稼働実績 | 設備の稼働状態・停止理由を記録する | 推奨 |
| WO-006 | 遅延アラート | 計画完了日を超過した工程を通知する | 推奨 |
| WO-007 | 進捗ダッシュボード | 全製造指示の進捗をリアルタイム表示 | 必須 |

---

## 機能詳細

### WO-001/002 工程着手・完了登録

**入力データ：**

| 項目 | 必須 | 説明 |
|------|------|------|
| 製造指示番号 | ○ | 対象の製造指示 |
| 工程コード | ○ | 作業対象工程 |
| 作業者 | ○ | 実施作業者（ログインユーザーまたはバーコードスキャン） |
| 設備コード | - | 使用設備 |
| 着手日時 | ○ | 実際の作業開始日時 |
| 完了日時 | ○（完了時） | 実際の作業完了日時 |
| 実績数量 | ○（完了時） | 完了した数量 |
| 不良数量 | - | 発生した不良数量 |
| 備考 | - | 特記事項 |

**入力方法：**

- 画面入力（タブレット）
- バーコード・QRコードスキャンによる製造指示・工程の読取

---

### WO-003 工程進捗照会

**表示内容：**

- 製造指示ごとの全工程の進捗状態
- 計画日程 vs. 実績日程の比較
- 工程別の完了率（数量ベース）

**進捗ステータス：**

| ステータス | 説明 |
|-----------|------|
| 未着手 | まだ着手していない |
| 着手中 | 作業中 |
| 完了 | 全数完了 |
| 遅延 | 計画完了日を過ぎても未完了 |
| 保留 | 問題発生等により一時停止 |

---

### WO-005 設備稼働実績

**停止区分：**

| 停止区分 | 説明 |
|---------|------|
| 計画停止 | 定期メンテナンス等の計画的停止 |
| 故障停止 | 設備故障による停止 |
| 段取り替え | 品種切替作業中 |
| 待機 | 材料待ち・指示待ち |
| その他 | 上記以外 |
