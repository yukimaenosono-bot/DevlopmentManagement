# 06. データ要件

## 6.1 主要エンティティ一覧

| エンティティ | 説明 | 分類 |
|------------|------|------|
| 品目マスタ | 製品・原材料・部品の基本情報 | マスタ |
| 取引先マスタ | 顧客・仕入先の基本情報 | マスタ |
| 工程マスタ | 製造工程の定義 | マスタ |
| 設備マスタ | 製造設備・機械の情報 | マスタ |
| 倉庫・棚マスタ | 倉庫・保管場所の定義 | マスタ |
| 作業者マスタ | 作業員・担当者情報 | マスタ |
| BOM（部品表） | 品目を構成する原材料・部品の構成 | マスタ |
| 工程ルーティング | 品目ごとの製造工程順序・標準時間 | マスタ |
| 検査基準マスタ | 品目・工程ごとの検査項目・合格基準 | マスタ |
| 受注 | 顧客からの注文情報 | トランザクション |
| 生産計画 | 期間別・品目別の生産計画 | トランザクション |
| 製造指示 | 製造指示ヘッダ・明細 | トランザクション |
| 工程実績 | 工程の着手・完了・実績情報 | トランザクション |
| 在庫 | 現在の在庫数量・ロット情報 | トランザクション |
| 入出庫履歴 | 在庫の入出庫トランザクション | トランザクション |
| 検査実績 | 検査結果・合否情報 | トランザクション |
| 不良情報 | 不良発生・処置情報 | トランザクション |
| 出荷指示 | 出荷指示ヘッダ・明細 | トランザクション |
| 出荷実績 | 実際の出荷情報 | トランザクション |

---

## 6.2 主要マスタデータ定義

### 品目マスタ

| 項目名 | データ型 | 必須 | 説明 |
|--------|---------|------|------|
| 品目コード | VARCHAR | ○ | 一意の品目識別コード |
| 品目名 | VARCHAR | ○ | 品目の正式名称 |
| 品目名（略称） | VARCHAR | - | 略称 |
| 品目区分 | CODE | ○ | 製品 / 半製品 / 原材料 / 消耗品 |
| ロット管理区分 | CODE | ○ | ロット管理あり / なし |
| シリアル管理区分 | CODE | ○ | シリアル管理あり / なし |
| 単位 | CODE | ○ | 個・kg・m 等 |
| 標準単価 | DECIMAL | - | 標準原価 |
| 有効期限管理 | BOOLEAN | ○ | 有効期限管理の有無 |
| 安全在庫数 | DECIMAL | - | 安全在庫の設定値 |
| 有効フラグ | BOOLEAN | ○ | 使用中 / 廃止 |

### BOM（部品表）

| 項目名 | データ型 | 必須 | 説明 |
|--------|---------|------|------|
| 親品目コード | VARCHAR | ○ | 上位品目 |
| 子品目コード | VARCHAR | ○ | 構成品目 |
| 必要数量 | DECIMAL | ○ | 親1単位あたりの必要数量 |
| 単位 | CODE | ○ | 数量の単位 |
| 有効開始日 | DATE | ○ | BOM有効期間の開始日 |
| 有効終了日 | DATE | - | BOM有効期間の終了日 |

---

## 6.3 ロット管理方針

| 対象 | 管理方式 | 説明 |
|------|---------|------|
| 原材料 | ロット管理 | 仕入先ロット番号で管理 |
| 仕掛品 | 製造ロット管理 | 製造指示と紐付くロット番号 |
| 完成品 | ロット / シリアル管理 | 品目設定による |

---

## 6.4 データ保持・アーカイブ方針

| データ種別 | 保持期間 | アーカイブ |
|-----------|---------|-----------|
| 製造実績・工程実績 | - 年 | - 年以降アーカイブ |
| 品質検査実績 | - 年（法規制に応じて） | - |
| 在庫履歴 | - 年 | - |
| 出荷実績 | - 年 | - |
| 操作ログ | - 年 | - |

---

## 6.5 採番ルール

| 対象 | 採番ルール（案） | 例 |
|------|--------------|-----|
| 製造指示番号 | MO-YYYYMMDD-NNNN | MO-20260307-0001 |
| 製造ロット番号 | LOT-YYYYMMDD-NNNN | LOT-20260307-0001 |
| 出荷指示番号 | SH-YYYYMMDD-NNNN | SH-20260307-0001 |

> ※ 採番ルールは業務要件に合わせて変更可
