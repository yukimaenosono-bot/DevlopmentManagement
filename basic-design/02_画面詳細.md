# 02. 画面詳細

## SCR-COM-002 ダッシュボード

| 項目 | 内容 |
|------|------|
| 対象ユーザー | 全ユーザー（表示内容はロールにより変化） |

| 要素 | 内容 | 更新方式 |

| 出荷予定 | 本日・翌日の出荷指示一覧 | 画面遷移時 |

## SCR-MO-002 製造指示発行

| 項目 | 内容 |
|------|------|
| 対象ユーザー | 生産管理担当 |
| 端末 | PC |

### 入力項目

| 項目 | 必須 | 型 | 説明 |
|------|------|-----|------|
| 製品コード | ○ | 検索テキスト | 品目マスタから選択 |

| 製造開始予定日 | ○ | 日付 | |
| 完了予定日 | ○ | 日付 | |

| 参照図番 | - | テキスト | |
| 作業指示コメント | - | テキスト（複数行） | |

### 工程明細

| 項目 | 説明 |
|------|------|

| 工程コード / 工程名 | マスタ参照 |

## SCR-SH-003 出荷実績登録

| 項目 | 内容 |
|------|------|
| 対象ユーザー | 出荷担当 |
| 端末 | PC・ハンディターミナル |

### 入力項目

| 項目 | 必須 | 説明 |
|------|------|------|
| 出荷指示番号 | ○ | バーコードスキャンまたは手入力 |
| 出荷日 | ○ | |
| 出荷数量 | ○ | 指示数量以内 |
| 出荷ロット番号 | ○ | 在庫ロットから選択（FIFO優先表示） |
| 運送会社 / 送り状番号 | - | |

### バリデーション

- 出荷数量 ≤ 出荷指示数量
- 完成品検査が「合格」であること（BR-SH-001）
- 在庫区分が「通常在庫」であること
